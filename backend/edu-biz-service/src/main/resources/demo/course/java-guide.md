# Java 课程导学

## 学习路线

1. 安装 JDK 21，确认 `java --version` 与 `javac --version` 可用。
2. 理解源文件、编译产物与 JVM 的关系。
3. 掌握变量、分支、循环、方法、类与对象。
4. 使用集合、异常和单元测试完成小型课程项目。

## 第一个程序

```java
public class HelloEducation {
    public static void main(String[] args) {
        System.out.println("Hello, smart education!");
    }
}
```

验收时应能说明编译与运行命令，并能解释 `main` 方法为什么是程序入口。
