# 高等数学公式表

## 极限

- `lim(x→0) sin(x) / x = 1`
- `lim(x→∞) (1 + 1/x)^x = e`
- 极限存在时满足和、差、积、商的四则运算法则，其中商的分母极限不能为 0。

## 导数

- `(x^n)' = n x^(n-1)`
- `(sin x)' = cos x`
- `(cos x)' = -sin x`
- `(e^x)' = e^x`
- 复合函数：`[f(g(x))]' = f'(g(x))g'(x)`

## 应用

当 `f'(x) > 0` 时函数在相应区间单调递增；当 `f'(x) < 0` 时单调递减。极值候选点需要结合驻点、不可导点及区间端点共同判断。
