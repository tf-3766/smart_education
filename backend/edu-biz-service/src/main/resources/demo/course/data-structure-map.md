# 数据结构思维导图

## 线性结构

- 线性表
  - 顺序表：连续存储，按下标访问为 O(1)，中间插入和删除通常为 O(n)。
  - 链表：节点离散存储，定位通常为 O(n)，已知节点位置后的插入和删除为 O(1)。
- 栈：先进后出，典型操作为 push、pop、peek，可用于表达式求值和深度优先搜索。
- 队列：先进先出，典型操作为 offer、poll，可用于任务调度和广度优先搜索。

## 树与图

- 二叉树遍历
  - 前序：根 → 左 → 右
  - 中序：左 → 根 → 右
  - 后序：左 → 右 → 根
- 图遍历
  - BFS 使用队列，适合无权图最短路径。
  - DFS 使用递归或栈，适合连通性、拓扑结构探索。

## 快速排序分区

选择基准值后，将小于基准的元素放在左侧、大于基准的元素放在右侧，再递归处理两个子区间。平均时间复杂度为 O(n log n)，极端不平衡分区时最坏为 O(n²)。
