# smart_education

- `docs/api-contract-platform.md`：前后端接口契约。
- `docs/database-schema-design.md`：数据库脚本设计说明。
- `在线教育辅助教学系统_项目设计文档_第一组.md`：项目设计文档 Markdown 源。
